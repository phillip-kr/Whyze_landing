The Problem

지금 당신의 투자 기록에는
무엇이 빠져 있나요?

수익이 났어도 '왜 맞았는지'를 모른다면, 다음 전략은 여전히 운에 의존합니다.

수익의 이유를 기록하지 않는다
대부분의 투자자는 매매 결과만 기록합니다. 어떤 논리로 진입했는지, 그 논리가 맞았는지는 추적되지 않습니다.
"수익이 났는데 왜 맞았는지 설명 못 하겠어요."
전략과 실행이 분리되지 않는다
전략이 나빴는지, 실행이 나빴는지 구분되지 않으면 어디를 개선해야 할지 알 수 없습니다. 두 가지가 뭉뚱그려집니다.
"손해가 났는데 전략 문제인지 타이밍 문제인지 모르겠어요."
시그널은 보이지만 연결이 없다
뉴스, 데이터, 지표들은 넘쳐납니다. 하지만 그것이 '내 전략 가정'과 어떻게 연결되는지 체계가 없으면 노이즈와 다름없습니다.
"HBM 데이터 보는데 내 투자랑 무슨 관계인지 모르겠어요."
3-Way Attribution

결과를 3가지로 분해합니다

수익이든 손실이든, 원인은 언제나 전략 효과 + 실행 효과 + 시장 효과의 합입니다. Whyze는 이 세 가지를 명확히 분리합니다.

How It Works

투자 판단의 전 과정을
하나의 OS로 운영합니다

설정 → 추적 → 기록 → 학습의 사이클이 반복될수록, 당신의 전략 역량은 복리로 쌓입니다.

전략 가정 구조화
투자 아이디어를 검증 가능한 가정으로 분해합니다. 각 가정은 프록시 시그널과 자동으로 연결되어 실시간 추적됩니다.
프록시 시그널 추적
전략 가정마다 맞춤 KPI를 설정하고 변화를 모니터링합니다. 뉴스가 아닌 데이터 기반으로 전략의 유효성을 판단합니다.
트레이드 로그 연결
매수·매도 기록을 전략과 연결합니다. 어떤 가정에 따라 진입했는지, 어떤 시그널 변화에 반응했는지 맥락이 보존됩니다.
귀인 분석 리포트
전략 종료 후 전략 효과·실행 효과·시장 효과를 자동 분해합니다. 다음 전략에 반영할 구체적 개선 포인트를 제시합니다.
